# 🔥 Python Error Solver v3.0

A Gradio web app that catches Python errors, pinpoints the exact line, explains in easy Urdu/English, and gives you auto-fixed code instantly.

### ✨ Features
- 🎯 **Smart Detection**: IndexError, NameError, ZeroDivisionError, KeyError, TypeError
- 📍 **Line Number**: Batata hai error kis line pe hua
- 💡 **Easy Explanation**: Asaan zuban mein error ka reason
- 🔧 **Auto-Fix**: One-click corrected code 
- 📜 **Error History**: Last 5 errors track karta hai
- 📁 **File Upload**: Direct .py file upload karo

### 🚀 Run Locally
```bash
pip install gradio
python Solver.py.py
